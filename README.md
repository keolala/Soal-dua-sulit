Deskripsi Program

Program ini ditulis dalam bahasa Java dan digunakan untuk melakukan penjumlahan dua buah matriks berukuran sama (memiliki jumlah baris dan kolom yang sama).

Program akan:

Meminta jumlah baris dan kolom dari pengguna.

Meminta elemen-elemen dari Matriks A dan Matriks B.

Menjumlahkan kedua matriks elemen demi elemen.

Menampilkan hasil penjumlahan dalam bentuk Matriks Hasil (A + B).

Cara Kerja Program

Pengguna diminta memasukkan jumlah baris dan kolom matriks.

Program membuat tiga buah array dua dimensi:

matriksA â†’ menyimpan elemen matriks pertama.

matriksB â†’ menyimpan elemen matriks kedua.

hasil â†’ menyimpan hasil penjumlahan kedua matriks.

Pengguna memasukkan nilai untuk setiap elemen matriks A dan B menggunakan dua loop bersarang (for).
